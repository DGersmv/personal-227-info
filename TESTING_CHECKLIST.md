# Чеклист для тестирования

## Что можно протестировать сейчас

### 1. Аутентификация
- [ ] Вход через временную форму (test@example.com / test123)
- [ ] Вход как Проектировщик (designer@test.com / test123)
- [ ] Вход как Строитель (builder@test.com / test123)
- [ ] Вход как Заказчик (customer@test.com / test123)
- [ ] Выход из системы (должен перенаправлять на страницу входа)

### 2. Dashboard
- [ ] Отображение Dashboard для каждой роли
- [ ] Карточки навигации (Объекты, Проекты, Загрузки)
- [ ] Панель для Проектировщика с быстрыми ссылками
- [ ] Правильное отображение роли пользователя

### 3. Объекты
- [ ] Создание объекта (как Заказчик)
- [ ] Просмотр списка объектов
- [ ] Просмотр детальной страницы объекта
- [ ] Вкладки на странице объекта (Обзор, Проекты, Фото, Панорамы, Документы, Сообщения)
- [ ] Проверка прав доступа (Заказчик видит только свои объекты)

### 4. Проекты
- [ ] Создание проекта (как Проектировщик)
- [ ] Просмотр списка проектов на объекте
- [ ] Просмотр детальной страницы проекта
- [ ] Создание этапов проекта
- [ ] Изменение статуса этапа (как Строитель)
- [ ] Удаление этапа (как Проектировщик)

### 5. Загрузки (для Проектировщиков)
- [ ] Просмотр списка загрузок
- [ ] Фильтры: по типу (Плагины, Программы, Шаблоны, Библиотеки)
- [ ] Фильтры: по цене (Все, Бесплатные, Платные)
- [ ] Фильтры: по программам (Все, Archicad, Revit, Российские)
- [ ] Поиск по названию и описанию
- [ ] Добавление новой загрузки
- [ ] Загрузка файла при создании загрузки
- [ ] Скачивание бесплатных загрузок
- [ ] Попытка скачать платную загрузку (должно требовать оплату)

### 6. Назначение пользователей на объекты
- [ ] Проверка, что Проектировщик видит назначенные объекты
- [ ] Проверка, что Строитель видит назначенные объекты
- [ ] Создание проекта на назначенном объекте (как Проектировщик)

## Что нужно протестировать после реализации

### 7. Фото (частично готово)
- [ ] Просмотр страницы фото объекта
- [ ] Фильтрация по папкам (когда будут реализованы папки)
- [ ] Загрузка фото (когда будет реализована)

### 8. Панорамы (в планах)
- [ ] Просмотр панорам
- [ ] Загрузка панорам

### 9. Документы (в планах)
- [ ] Загрузка документов
- [ ] Просмотр документов
- [ ] Статус оплаты

### 10. Сообщения (в планах)
- [ ] Переписка по объекту
- [ ] Уведомления

## Известные ограничения

1. Загрузка файлов для фото пока не реализована (API возвращает 501)
2. Оплата через insales пока не настроена (будет сообщение)
3. Портфолио добавлено в план, но еще не реализовано
4. Документы и сообщения в планах

## Рекомендуемый порядок тестирования

1. **Базовый функционал:**
   - Вход/выход
   - Dashboard
   - Навигация

2. **Объекты и проекты:**
   - Создание объекта
   - Создание проекта
   - Управление этапами

3. **Загрузки:**
   - Просмотр и фильтры
   - Добавление загрузки
   - Скачивание

4. **Права доступа:**
   - Проверка видимости объектов для разных ролей
   - Проверка прав на создание/редактирование
