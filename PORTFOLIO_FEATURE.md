# Портфолио для Проектировщиков и Строителей

## Обзор

Каждый Проектировщик и Строитель может создать и настроить свое портфолио для демонстрации своих работ и навыков.

## Функционал

### 1. Редактирование карточки портфолио

#### Основная информация:
- **Заголовок портфолио** - название портфолио
- **Описание** - краткое описание портфолио
- **Биография** - подробная информация "О себе"
- **Аватар** - фото профиля
- **Обложка** - фоновое изображение портфолио
- **Цвет фона** - выбор цвета фона (hex)

#### Контакты:
- **Телефон** - контактный телефон
- **Веб-сайт** - личный сайт или страница
- **Социальные сети** - ссылки на:
  - LinkedIn
  - Instagram
  - VK
  - Telegram
  - Другие платформы

#### Настройки видимости:
- **Публичное портфолио** - сделать портфолио доступным по публичной ссылке
- **Показывать email** - отображать email в публичном портфолио
- **Показывать телефон** - отображать телефон в публичном портфолио

### 2. Управление проектами в портфолио

#### Добавление проектов:
- Выбор проектов из реальных проектов пользователя
- Или создание отдельной записи проекта для портфолио
- Загрузка главного изображения проекта
- Описание проекта
- Категория проекта
- Теги для поиска

#### Организация проектов:
- Изменение порядка отображения (drag & drop)
- Скрытие/показ проектов
- Редактирование информации о проекте
- Удаление проектов

### 3. Просмотр портфолио

#### Для владельца:
- Полный доступ к редактированию
- Статистика просмотров
- Предпросмотр публичного вида

#### Публичный просмотр:
- Доступ по ссылке `/portfolio/[userId]`
- Отображение только публичной информации
- Фильтрация по категориям и тегам
- Контактная информация (если разрешено)

## Права доступа

### Проектировщик (DESIGNER):
- ✅ Создание и редактирование своего портфолио
- ✅ Добавление проектов в портфолио
- ✅ Управление видимостью
- ✅ Просмотр статистики

### Строитель (BUILDER):
- ✅ Создание и редактирование своего портфолио
- ✅ Добавление проектов в портфолио
- ✅ Управление видимостью
- ✅ Просмотр статистики

### Заказчик (CUSTOMER):
- ❌ Не может создавать портфолио
- ✅ Может просматривать публичные портфолио
- ✅ Может искать проектировщиков и строителей по портфолио

### Администратор (ADMIN):
- ✅ Полный доступ ко всем портфолио
- ✅ Может редактировать любое портфолио
- ✅ Может управлять видимостью

## Структура данных

### Portfolio Model:
```prisma
model Portfolio {
  id, userId (unique)
  title, description, bio
  phone, website, socialLinks (JSON)
  avatarUrl, coverImageUrl, backgroundColor
  isPublic, showEmail, showPhone
  viewCount
  createdAt, updatedAt
}
```

### PortfolioProject Model:
```prisma
model PortfolioProject {
  id, portfolioId, projectId (optional)
  title, description, imageUrl
  category, tags (JSON)
  orderIndex, isVisible
  createdAt, updatedAt
}
```

## API Endpoints

### Управление портфолио:
- `GET /api/portfolio` - получить свое портфолио
- `PUT /api/portfolio` - обновить свое портфолио
- `GET /api/portfolio/[userId]` - получить публичное портфолио пользователя

### Управление проектами:
- `GET /api/portfolio/projects` - список проектов в портфолио
- `POST /api/portfolio/projects` - добавить проект
- `PUT /api/portfolio/projects/[id]` - обновить проект
- `DELETE /api/portfolio/projects/[id]` - удалить проект
- `PUT /api/portfolio/projects/reorder` - изменить порядок проектов

### Загрузка файлов:
- `POST /api/portfolio/upload` - загрузить аватар или обложку

## Страницы

### `/portfolio`
- Страница управления своим портфолио
- Редактирование карточки
- Управление проектами
- Статистика

### `/portfolio/edit`
- Форма редактирования карточки портфолио
- Все поля для настройки

### `/portfolio/[userId]`
- Публичная страница портфолио
- Отображение проектов
- Контактная информация (если разрешено)
